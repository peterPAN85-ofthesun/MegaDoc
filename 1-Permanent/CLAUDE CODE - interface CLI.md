# CLAUDE CODE - interface CLI

Claude Code est l'interface en ligne de commande officielle de [[CLAUDE - assistant IA Anthropic|Claude]], permettant d'interagir avec l'IA directement depuis le terminal. Elle offre des outils spécialisés pour manipuler des fichiers, exécuter des commandes système, rechercher dans le code, et automatiser des tâches complexes via des agents spécialisés.

La force de Claude Code réside dans son architecture à outils multiples : Read/Write/Edit pour les fichiers, Grep/Glob pour la recherche, Bash pour l'exécution shell, et un système d'agents pour déléguer des tâches complexes multi-étapes. L'interface supporte également l'automatisation via Slash Commands personnalisés et Hooks événementiels configurables.

## Exemples

**Outils principaux** :
- Manipulation fichiers : [[CLAUDE CODE : [Read] - lecture fichiers multimédia|Read]], [[CLAUDE CODE : [Write] - écriture fichiers|Write]], [[CLAUDE CODE : [Edit] - remplacement chaînes|Edit]]
- Recherche : [[CLAUDE CODE : [Grep] - recherche ripgrep|Grep]], [[CLAUDE CODE : [Glob] - recherche fichiers pattern|Glob]]
- Système : [[CLAUDE CODE : [Bash] - exécution shell|Bash]]
- Web : [[CLAUDE CODE : [WebFetch] - récupération web|WebFetch]], [[CLAUDE CODE : [WebSearch] - recherche temps réel|WebSearch]]
- Organisation : [[CLAUDE CODE : [TodoWrite] - gestion tâches|TodoWrite]], [[CLAUDE CODE : [AskUserQuestion] - questions interactives|AskUserQuestion]]

**Automatisation** : [[CLAUDE CODE - Slash Commands|Slash Commands]], [[CLAUDE CODE - Hooks personnalisés|Hooks]], [[CLAUDE CODE - MCP Servers|MCP]]

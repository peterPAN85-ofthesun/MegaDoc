# Notes atomiques - Principe fondamental du Zettelkasten

> [!abstract] Concept
> Une note atomique contient une seule idée, complète et compréhensible de manière autonome.

## Explication

Le principe d'atomicité signifie qu'une note doit :
1. **Traiter un seul concept** - Ne pas mélanger plusieurs idées
2. **Être autonome** - Compréhensible sans lire d'autres notes (même si des liens enrichissent)
3. **Être complète** - L'idée est développée suffisamment
4. **Être réutilisable** - Peut être liée dans différents contextes

L'atomicité permet de créer des connexions flexibles entre idées. Une note non-atomique limite les possibilités de liens et de réutilisation.

## Exemples

✅ **Bon (atomique)** :
- "NAT - Network Address Translation"
- "Pointeurs en C"
- "Git rebase vs merge"

❌ **Mauvais (non-atomique)** :
- "Cours jour 1 - Réseau" (trop large, multiples concepts)
- "Trucs et astuces Git" (mélange de concepts distincts)
